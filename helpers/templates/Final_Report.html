Segmentation Testing

The network segment defined within the Scope of Work as a target IPv4 range for segmentation testing can be reached from a network device hosted within a non-CDE Network segment. A tool named Nmap was used to port scan the target IPv4 ranges within the Scope's segments. Hosts within the Scope reported open ports exposed to the non-CDE network segment. As a result, a user or an attack with initial access into the non-CDE network could potentially move laterally in to the affected CDE segment and possibly compromise credentials, customer data, or other sensitive secrets.
